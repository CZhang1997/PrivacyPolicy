Markdown Viewer Privacy Policy

Last updated: June 2, 2026

Short version: Markdown Viewer renders your markdown files locally in your browser. Your document contents are not uploaded for viewing. Optional Pro billing uses ExtensionPay for hosted checkout, login, purchase restoration, and license verification.

Single Purpose

Markdown Viewer renders markdown (.md) files with rich formatting including syntax highlighting, diagrams, and math equations, and provides a folder browser for organizing local markdown documents.

Information the Extension Stores Locally

The extension uses Chrome local storage and IndexedDB to save app state on your device, including:

  • User preferences such as theme, language, font scale, and folder scan depth.
  • Viewing history, pinned files, recent tabs, and folder metadata.
  • Folder handles for folders you explicitly choose through the browser folder picker.
  • A cached billing or license status to keep the interface responsive.

This information is stored locally in your browser and is used to provide the extension's markdown viewing and folder browsing features.

Markdown File Contents

Markdown files are processed locally in the browser for rendering, search, table of contents generation, and export features. Markdown document contents are not uploaded to our servers for viewing.

Optional Pro Billing

Markdown Viewer uses ExtensionPay for optional Pro features. When you start a trial, purchase, sign in, restore a purchase, or refresh license status, ExtensionPay may process billing-related information such as your email address, payment status, subscription status, and purchase details.

The extension uses https://extensionpay.com/* only for hosted checkout, trial, login, purchase restoration, and license verification. Markdown document contents are not sent to ExtensionPay for viewing.

Permissions

  • storage: Saves preferences, folder metadata, viewing history, pins, recent tabs, and billing/license cache locally.
  • file access: Allows the extension to render local markdown files that you open in Chrome and browse folders you explicitly choose.
  • extensionpay.com host access: Supports optional Pro checkout, login, purchase restoration, and license verification.

Analytics and Tracking

Markdown Viewer does not collect document analytics, browsing analytics, or telemetry about the contents of your markdown files.

Data Sharing

Markdown document contents are not sold, shared, or uploaded for viewing. Billing-related information is handled by ExtensionPay only when you use optional Pro billing features.

Contact

For privacy questions or support, contact the extension developer through the support link provided on the Chrome Web Store listing.
